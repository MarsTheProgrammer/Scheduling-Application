Description:

This application was built for an international company and was design for the primary purpose of scheduling appointments for clients. There are customers and contacts. Each have their own functionality and purpose. The customer has the ability to schedule appointments with various contacts, assuming there are within business hours, and do not conflict with other appointments at that time.

The application has an appointment reminder system that prompts the user if there is an appointment within the next 15 minutes. If there is an appointment, the notication with prompt with user with information regarding time, location, who, and appointment ID.

There is bi-lingual functionality for English and French.
